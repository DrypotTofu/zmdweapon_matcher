# 武器匹配系统

一个帮助你找到完美匹配武器的工具。

## 在线访问

[点击这里访问网站](https://your-username.github.io/JiZhiPiPei/weapon_matcher.html)

## 功能

- 🎯 根据基质属性匹配武器
- 📱 完美适配移动端
- ✨ 美观的用户界面
- 🚀 无需安装，打开即用

## 本地使用

直接用浏览器打开 `weapon_matcher.html` 文件即可。

## 属性说明

### 基础属性
敏捷、力量、意志、智识、主能力

### 附加属性
攻击、生命、物理伤害、灼热伤害、电磁伤害、寒冷伤害、自然伤害、暴击率、源石技艺、治疗效率、终结技效率、法术伤害

### 技能属性
强攻、压制、追袭、粉碎、昂扬、巧技、残暴、附术、医疗、切骨、迸发、夜幕、流转、效益
